# IFAWL对象术语命名约定

这是一份开发人员命名约定文稿，旨在统一不同开发者在代码中对不同对象的称呼

## 终焉结 Al-al(s)

指舰船上通过qwe触发功能的对象

Al_general

my_ship.load_al()

my_ship.get_equivalent_shelter_from_als()

## 合同 Contract-contract(s)

指星际市场中可供交易的对象

Contract_manager

contract_manager.print_all_contracts()

## 树 Tree-tree(s)

指出现在空间站、战斗时、结算画面的树状打印结构

StationTreesManager

tree.print_self()

## 词条 Entry-entry(-ies)

指出现在各色战斗中的、为敌方进行增强的对象

EntryManager

entry_manager.generate_entry_summary_lines()

## 协议 Card-card(s)

指出现在无尽模式中的、为我方进行增强的对象

## 事件 Ocp-ocp(s)

指随即出现在战斗过程中、具有剧情打印和各色增减益效果的对象

ocp_manager.current_ocp
