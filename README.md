## 战斗⚔️
战斗是获取资源的最佳手段。IFAWL的战斗由以下成分构成：
### 摇骰子 🎲
IFAWL的回合单位是“天”。每天是你行动还是敌方行动是随机决定的。骰子通过动态概率调节来保证公平性，同时避免某一方连续行动过多。我方行动的日子被称为**航行日** 。
### 护盾 🛡️
IFAWL中的**血量**。在战场显示中显示为“-----”，是敌我双方的生存保障。当护盾被清空而没有急救性质的终焉结时，再受击一次即判定失败。即便拥有护盾，若单次攻击伤害大于护盾数量而没有急救，也会判定失败。航行日输入[2]来回充一层护盾。许多w系终焉结也能回充或强化护盾，抑或提供急救能力。
### 导弹 🚀
IFAWL中的**平A手段**。航行日输入[0或space]来装载一枚导弹。已装载的导弹显示为“[]”。航行日输入1来打出一枚导弹，造成1伤害。许多q系终焉结提供更猛烈的伤害，或对普通导弹伤害、上弹速度提供加成。
### 终焉结 🗡️
终焉结是IFAWL战斗体验的重要组成部分，分为q|w|e三类。它们的详细技能描述和使用方式可参见终焉结选择界面的文本。下面分类对它们进行简要介绍。
#### q系终焉结|主武器
激活其功能多数需要输入q，详见其描述（w|e同理）
- 提供猛烈的爆发伤害（湾铃#30 “湾区铃兰”饱和式蜂巢突击粒子炮）
- 或对导弹伤害提供加成（维多利亚#8 “维多利亚”加农炮式导弹发射架）
- 或高速发射现有导弹（暴雨#15 “暴雨”全自动射手导弹发射台）

#### w系终焉结|生存位
- 为护盾回充量提供加成（如修械师#9“修械师”常规维修小队吊舱）
- 提供空盾时急救（白金#6“白金”紧急维修小队前线安全屋）、
- 强化硬化护盾（信风#14“信风”全结晶护盾防御系统）
- 立即生成护盾（白鲟#31“白鲟”薄壁共振护盾再生平台）

#### e系终焉结|战术区
e系终焉结是最为人杰地灵的。它能提供极其多样化的战斗优势，包括但不限于：
- 提供伤害（风行者#3“风行者”战术巡飞弹）
- 削弱敌方（奶油#7“奶油”黑客老姐）
- 控制敌方行动（眠雀#26“眠雀”麻醉性致幻剂发射器）
- 为我方提供减伤或闪避（瞳猫#27“瞳猫”反应堆尾流推进器）
- 操纵任务时间、加速需冷却终焉结的冷却过程（青鹄#35“青鹄”时空连续体空洞构造反应堆）

#### 关于无尽模式

在无尽模式中，指挥官将携带低级终焉结进入对局，并接受多波次的敌人袭击。

在波次之间，指挥官可以选择逃离战场并回到空间站，或是继续往战场深处进发，迎击更为强大的敌人。战斗波次越高，敌方初始导弹和护盾就越高，同时将得到更加强大的词条

**协议**将在波次之间出现，帮助指挥官抵消敌方的优势。典型的协议有：

> 紧急回站 护盾/导弹 ++

而**最重要的协议**是

>**终焉结支援** 升级舰船的终焉结

#### 升级终焉结

指挥官在开局时可以任意选择0~2级的终焉结，使用q/w/e即可调整

通过在两次战斗之间的**终焉结支援**协议，指挥官可以把q/w/e的最大终焉结等级提升2级，来使用更强大的终焉结抗击敌方
### 全新终焉结上线

#### 版本大C “星尘”非自动制导正光锥粒子流 #37/q
- 极致优雅·极致精准·极致射击体验·粒子炮平台诚意之作
- 在航行日输入q进入预备状态，轻按[enter]进入锁定模式
- 锁定时，当"|"符号运动至"[ + ]"正下方时，按下[ctrl+C]锁定敌方并发射粒子炮
- 根据精确度消耗粒子匣并造成消耗粒子匣1.5倍的伤害
- [星尘无尽]无尽/深空探索模式下，溢出的伤害按照一定比例转化回粒子匣，以供下一轮使用

星尘是首个含有qte机制的终焉结。指挥官需要在特定的时机按下ctrl+C来发射粒子炮，瞬间消耗大量粒子匣造成你**想都不敢想**的**巨大**伤害(DMG_max = **15.0**)，尤其适合抗击无尽模式后期的重甲敌人。同时，星尘对无尽模式有特殊的优化技能，溢出的伤害将按照一定比例转化回粒子匣给下一轮使用，这使得星尘成为无尽模式后期的“指定”终焉结之一。

#### “西岭”高速传统近防炮防空体系 #36/e
- 极其传统的大口径速射防空炮，可以自动防空拦截敌方导弹，也可以用于主动进攻
- 默认模式为自动防空模式。有概率拦截敌方的普通攻击
- 输入36切换至主动进攻模式。每个航行日获得一次弹雨扫射机会，共四十枚速射弹药连续打出，命中可造成1伤害

50%的常态闪避是多少猫的一辈子，你说是吧瞳猫？

#### “冷水”超临界冰晶护盾注入剂 #40/e
- 来自岩河军工的护盾增强体系，将水注入结晶化护盾的夹层之间结冰使护盾得以强化
- 输入w将一面护盾强化为冷水护盾，每层冷水护盾有一定概率完全拦下敌方的攻击
- 若不能拦下，该层冷水盾破碎，使敌方攻击减一并交给下一层冷水盾继续处理
- [应力结霜]冷水盾破碎将对敌方造成1伤害

设计冷水时我们曾经考虑过把她放在w区，结果发现概率护盾作为生存的手段显然是不及格的。作为战术区护盾强化终焉结，冷水可以和许多w区的伙伴形成紧密的配合，通过概率盾和次数盾/强化盾/多层盾相互叠加获得生存优势。

### 不正常现象
这些现象是**不正常**的。遇到它们时，请联系我们。
- 任何游戏崩溃
- 终焉结在两次战斗之间未能重置自身状态，如冷却冷到下一把开始还没冷却完
- 仓库物品数量异常，出现负数或不能正常增减
- 战斗过程中出现未携带的终焉结的话语
- 舰船名字是空白或无法正确命名舰船
- 自动驾驶出现错误操作
- 终焉结行为与描述不匹配（如大奶油的小奶油偷到导弹之后不能爆炸）
