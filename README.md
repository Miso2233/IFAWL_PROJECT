**诗岸#21**

诗岸是一位w系急救终焉结

- 携带诗岸时，按下2护盾充能将进行液态护盾充注
- 当舰船护盾受到致命伤害，消耗一层液态护盾并将舰船护盾补全至一层
- 若同时持有的液态护盾过多，有概率意外全部固化为普通护盾，失去急救能力
- 按下w主动将所有液态护盾以1.5倍转化为普通护盾

## 战斗
战斗是获取资源的最佳手段。IFAWL的战斗由以下成分构成：
### 摇骰子
IFAWL的回合单位是“天”。每天是你行动还是敌方行动是随机决定的。骰子通过动态概率调节来保证公平性，同时避免某一方连续行动过多。我方行动的日子被称为**航行日** 。
### 护盾
IFAWL中的**血量**。在战场显示中显示为“-----”，是敌我双方的生存保障。当护盾被清空而没有急救性质的终焉结时，再受击一次即判定失败。即便拥有护盾，若单次攻击伤害大于护盾数量而没有急救，也会判定失败。航行日输入[2]来回充一层护盾。许多w系终焉结也能回充或强化护盾，抑或提供急救能力。
### 导弹
IFAWL中的**平A手段**。航行日输入[0或space]来装载一枚导弹。已装载的导弹显示为“[]”。航行日输入1来打出一枚导弹，造成1伤害。许多q系终焉结提供更猛烈的伤害，或对普通导弹伤害、上弹速度提供加成。
### 终焉结
终焉结是IFAWL战斗体验的重要组成部分，分为q|w|e三类。它们的详细技能描述和使用方式可参见终焉结选择界面的文本。下面分类对它们进行简要介绍。
#### q系终焉结|主武器
激活其功能多数需要输入q，详见其描述（w|e同理）
- 提供猛烈的爆发伤害（湾铃#30 “湾区铃兰”饱和式蜂巢突击粒子炮）
- 或对导弹伤害提供加成（维多利亚#8 “维多利亚”加农炮式导弹发射架）
- 或高速发射现有导弹（暴雨#15 “暴雨”全自动射手导弹发射台）

#### w系终焉结|生存位
- 为护盾回充量提供加成（如修械师#9“修械师”常规维修小队吊舱）
- 提供空盾时急救（白金#6“白金”紧急维修小队前线安全屋）、
- 强化硬化护盾（信风#14“信风”全结晶护盾防御系统）
- 立即生成护盾（白鲟#31“白鲟”薄壁共振护盾再生平台）

#### e系终焉结|战术区
e系终焉结是最为人杰地灵的。它能提供极其多样化的战斗优势，包括但不限于：
- 提供伤害（风行者#3“风行者”战术巡飞弹）
- 削弱敌方（奶油#7“奶油”黑客老姐）
- 控制敌方行动（眠雀#26“眠雀”麻醉性致幻剂发射器）
- 为我方提供减伤或闪避（瞳猫#27“瞳猫”反应堆尾流推进器）
- 操纵任务时间、加速需冷却终焉结的冷却过程（青鹄#35“青鹄”时空连续体空洞构造反应堆）

#### 终焉结仓库
终焉结是指挥官仓库资产的一部分。在[z]访问仓库时可以看到。

对于仓库里没有的非免费终焉结，指挥官可以将它们装载到船上，但是这样会使得舰船不能离站。

只有舰船上装配的终焉结都在仓库里有存货
