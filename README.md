## 欢迎来到新伊甸 👋
欢迎游玩回合制策略游戏新伊甸命运：进攻战线！在这片充满了奇迹的空域，你将得到下列行为的完整权限：
### 在回合制战斗中和敌方体会人类智斗的巅峰 🎆
- 管理护盾和弹药，击毁敌方，获取物资与信用点赏金
- 启用船上的**终焉结**，获取巨大战斗优势
- 若被敌方击败，船上的终焉结将会**全部损毁**
### 访问星域市场并购置装备 💷
- 签署易物合同，互相交换**八种基本物资**
- 签署货品合同使用**信用点**购置物资，或出售物资获取信用点
- 签署工业合同，通过物资和信用点获取**终焉结**
- 签署金融合同进行风险投资赚取信用点，成为华尔街之狼

### 配置终焉结 🗡️
- 在空间站里，把终焉结安装上船
- 终焉结分为q区|主武器 w区|生存位 e区|战术区，共有0到8级九个级别
- 终焉结是船上最重要的装备，直接决定了战斗的成败
- 设定上来说，每一个终焉结都由一套机械和一（多）位机师构成
- IFAWL将提供上千种终焉结组合，每一种组合都有着自己的风格

## 战斗⚔️
战斗是获取资源的最佳手段。IFAWL的战斗由以下成分构成：
### 摇骰子 🎲
IFAWL的回合单位是“天”。每天是你行动还是敌方行动是随机决定的。骰子通过动态概率调节来保证公平性，同时避免某一方连续行动过多。我方行动的日子被称为**航行日** 。
### 护盾 🛡️
IFAWL中的**血量**。在战场显示中显示为“-----”，是敌我双方的生存保障。当护盾被清空而没有急救性质的终焉结时，再受击一次即判定失败。即便拥有护盾，若单次攻击伤害大于护盾数量而没有急救，也会判定失败。航行日输入[2]来回充一层护盾。许多w系终焉结也能回充或强化护盾，抑或提供急救能力。
### 导弹 🚀
IFAWL中的**平A手段**。航行日输入[0或space]来装载一枚导弹。已装载的导弹显示为“[]”。航行日输入1来打出一枚导弹，造成1伤害。许多q系终焉结提供更猛烈的伤害，或对普通导弹伤害、上弹速度提供加成。
### 终焉结 🗡️
终焉结是IFAWL战斗体验的重要组成部分，分为q|w|e三类。它们的详细技能描述和使用方式可参见终焉结选择界面的文本。下面分类对它们进行简要介绍。
#### q系终焉结|主武器
激活其功能多数需要输入q，详见其描述（w|e同理）
- 提供猛烈的爆发伤害（湾铃#30 “湾区铃兰”饱和式蜂巢突击粒子炮）
- 或对导弹伤害提供加成（维多利亚#8 “维多利亚”加农炮式导弹发射架）
- 或高速发射现有导弹（暴雨#15 “暴雨”全自动射手导弹发射台）

#### w系终焉结|生存位
- 为护盾回充量提供加成（如修械师#9“修械师”常规维修小队吊舱）
- 提供空盾时急救（白金#6“白金”紧急维修小队前线安全屋）、
- 强化硬化护盾（信风#14“信风”全结晶护盾防御系统）
- 立即生成护盾（白鲟#31“白鲟”薄壁共振护盾再生平台）

#### e系终焉结|战术区
e系终焉结是最为人杰地灵的。它能提供极其多样化的战斗优势，包括但不限于：
- 提供伤害（风行者#3“风行者”战术巡飞弹）
- 削弱敌方（奶油#7“奶油”黑客老姐）
- 控制敌方行动（眠雀#26“眠雀”麻醉性致幻剂发射器）
- 为我方提供减伤或闪避（瞳猫#27“瞳猫”反应堆尾流推进器）
- 操纵任务时间、加速需冷却终焉结的冷却过程（青鹄#35“青鹄”时空连续体空洞构造反应堆）
#### 战斗辅助面板
记忆终焉结们复杂的启动和搓招流程？
> 水银#5 q q  0 0 q 1 1——水银的协助挂弹连招
> 
> 湾区铃兰#30 q 2 2 0 q——湾区铃兰的牺牲护盾自增伤连招

完全没有必要！没有必要！战斗辅助面板来救你！

**战斗辅助面板将在战斗界面上显示，明确展示这个终焉结此时可以按下哪些按键，有什么功能**

例子：

>"[q]发射台开机|激活预备导弹", 
>
>"[自动攻击中]当日发射|[q]发射台关机|[0]辅助上弹",
>
>"[自动攻击中]次日发射|[q]发射台关机"

是暴雨#15分别在不同状态下的战斗辅助提示。

对于自己的每一个状态，终焉结都会展示可以操作的按键，指挥官只需专心于攻击和生存的平衡，无需劳神记忆终焉结的具体操作。
#### 终焉结仓库
终焉结是指挥官仓库资产的一部分。在[z]访问仓库时可以看到。

对于仓库里没有的非免费终焉结，指挥官可以将它们装载到船上，但是这样会使得舰船不能离站。

只有舰船上装配的终焉结都在仓库里有存货时，指挥官才能离开空间站执行任务

在基本对战模式中，词条将随着作战时间的推进逐渐出现。指挥官离站时携带的终焉结总等级越高，词条首次出现的日期越早，词条升级的频率越高

#### 战死之地
我们加入了战死之地模式，让诸位手持顶级终焉结的指挥官体会超高难交锋的乐趣。在空间站中输入c1来编辑战死之地的词条。你可以选择点亮一两条小试牛刀，也可以全部点亮挑战IFAWL战斗难度的巅峰。输入x离站后，输入1进入战死之地。此时敌人将带有你所选择的所有词条。战斗！前进！直到战死于深空！

### 全新终焉结上线

#### 版本大C “黎明维多利亚”六冲程混合增压燃烧导弹推进器 #39 @q
- 浅草寺全新一代导弹改进技术，由维多利亚#8全权接手开发
- 黎明维多利亚随着每一枚[0]装载的导弹自动充能，亦可通过[q]装弹并充能
- 充能级别达到5层时启用，为接下来五枚导弹提供伤害加成
- [黎明爆发]打出2枚导弹消耗至3层后，可以输入[q]消耗全部层数瞬间打出对应量导弹，造成极限伤害。同时，我方护盾将承受相当于发射导弹数量-1的伤害

作为VIII级终焉结，黎明维多利亚有着超高的输出强度。同时，通过牺牲生存能力换来的爆发性输出为指挥官提供了更多的战术选择。

#### 版本生存位“澈”高牺牲护盾锐化改装套件 #38 @w
- 以牺牲为核心的新一代护盾平台，拥有比风间浦复杂的模式，比信风更灵活的进攻防守策略
- 航行日输入w获得三层锋镝，随之对自己造成两次共计两点伤害
- 任何时候，在护盾小于一时，澈将依照现有锋镝最多补充护盾至二，并消耗对应层数锋镝
- [澄]模式|澈的正常模式|在敌我发动攻击时获得一层锋镝
- 若锋镝大于九，进入[寂]
- [寂]模式|澈的牺牲模式|我方发动攻击时消耗一层锋镝使伤害加一，直到锋镝小于五时退出[寂]

澈是我们开发的最为复杂的生存位终焉结之一。简单地说，锋镝量会随着双方攻击而积累，到达9点时进入进攻模式，此时我方攻击将导致锋镝量下降并获得增伤，直到5点时退出进攻模式。

推荐搭配**大奶油#24**食用

#### 版本生存位“风间浦”全平台战术安全港 #34 @w
- 双模式防御系统，提供从保守到激进的多种生存方案
- 以保守模式进入对局。在该模式下，所有回盾有概率加一。没有护盾时，这一概率上升至100%
- 输入w进入激进模式。这一模式下，舰船将不会死亡，持续至往后三个敌方日
- 激进模式结束后，回复与激进模式内受到总伤害等量的护盾，并进入五个航行日的冷却
- [烈风]在进入激进模式的瞬间，对敌方造成1伤害

风间浦是我们为带有护盾扣血机制的终焉结提供的最佳解决方案。在输入w进入的激进模式下，来自敌我的任何扣血都会被瞬间急救并记入风间浦的最终回血量中。退出激进模式的瞬间，风间浦将回充计入的等量护盾。无论是作为扣血输出的辅助，还是在超高难对局中生存下来，风间浦都是指挥官最好的伙伴。

推荐搭配**白夜#17 湾区铃兰#30 黎明维多利亚#39**食用

#### “白鲟”薄壁共振护盾再生平台 #31 @w
- 兼有上弹能力的护盾再生平台
- 在航行日输入w获得6层临时护盾并获得一枚弹药，每我方航行日流失一点
- 临时护盾耗竭后进入五个任意日的冷却

简约，而不简单。白鲟是IFAWL所有终焉结里HPR最高的，只是护盾流失的机制使得指挥官需要合理安排回盾窗口。

#### “青鹄”时空连续体空洞构造反应堆 #35 @e
- 来自唐朝的穿越者和她的时间机器（自称），使用无粒子引力源扭曲时间
- 在航行日输入e获得2充能，每天额外获得一充能，达到4充能时进入待命状态；待命期间输入e消耗所有充能与二枚弹药造成2伤害
- 待命时在敌方回合获得额外回合，获得一枚弹药和一层护盾，按下[q]\[w\][e]触发对应效果或清空冷却，随后减少4充能,退出待命状态
- [主观缓时]每天额外获得一充能；键入e能额外使任务天数减少一天
时光机……时光机。在敌方回合强行加入我方操作触发终焉结使得战斗节奏变得无比灵活。同时，清空终焉结冷却的能力，青鹄独树一帜。

- 任何游戏崩溃
