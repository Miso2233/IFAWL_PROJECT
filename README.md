## 战斗⚔️
战斗是获取资源的最佳手段。IFAWL的战斗由以下成分构成：
### 摇骰子 🎲
IFAWL的回合单位是“天”。每天是你行动还是敌方行动是随机决定的。骰子通过动态概率调节来保证公平性，同时避免某一方连续行动过多。我方行动的日子被称为**航行日** 。
### 护盾 🛡️
IFAWL中的**血量**。在战场显示中显示为“-----”，是敌我双方的生存保障。当护盾被清空而没有急救性质的终焉结时，再受击一次即判定失败。即便拥有护盾，若单次攻击伤害大于护盾数量而没有急救，也会判定失败。航行日输入[2]来回充一层护盾。许多w系终焉结也能回充或强化护盾，抑或提供急救能力。
### 导弹 🚀
IFAWL中的**平A手段**。航行日输入[0或space]来装载一枚导弹。已装载的导弹显示为“[]”。航行日输入1来打出一枚导弹，造成1伤害。许多q系终焉结提供更猛烈的伤害，或对普通导弹伤害、上弹速度提供加成。
### 终焉结 🗡️
终焉结是IFAWL战斗体验的重要组成部分，分为q|w|e三类。它们的详细技能描述和使用方式可参见终焉结选择界面的文本。下面分类对它们进行简要介绍。
#### q系终焉结|主武器
激活其功能多数需要输入q，详见其描述（w|e同理）
- 提供猛烈的爆发伤害（湾铃#30 “湾区铃兰”饱和式蜂巢突击粒子炮）
- 或对导弹伤害提供加成（维多利亚#8 “维多利亚”加农炮式导弹发射架）
- 或高速发射现有导弹（暴雨#15 “暴雨”全自动射手导弹发射台）

#### w系终焉结|生存位
- 为护盾回充量提供加成（如修械师#9“修械师”常规维修小队吊舱）
- 提供空盾时急救（白金#6“白金”紧急维修小队前线安全屋）、
- 强化硬化护盾（信风#14“信风”全结晶护盾防御系统）
- 立即生成护盾（白鲟#31“白鲟”薄壁共振护盾再生平台）

#### e系终焉结|战术区
e系终焉结是最为人杰地灵的。它能提供极其多样化的战斗优势，包括但不限于：
- 提供伤害（风行者#3“风行者”战术巡飞弹）
- 削弱敌方（奶油#7“奶油”黑客老姐）
- 控制敌方行动（眠雀#26“眠雀”麻醉性致幻剂发射器）
- 为我方提供减伤或闪避（瞳猫#27“瞳猫”反应堆尾流推进器）
- 操纵任务时间、加速需冷却终焉结的冷却过程（青鹄#35“青鹄”时空连续体空洞构造反应堆）

#### 关于联合狩猎 - 连接

联合狩猎是一种双人联机战死之地模式，不损耗终焉结，亦没有赏酬。

这一模式需要两位指挥官共同游玩。下面，我们把这两台设备分别称为长机和僚机，展示联机流程。

1. 请确保两台设备处于同一**局域网**或**虚拟局域网**下
2. 长机指挥官启动IFAWL，离站，选择**联合狩猎|长机**。这一操作将挂起服务端并持续监听僚机的连接请求
3. 长机和僚机指挥官使用**电话**、面对面**交谈**等手段取得联系
4. 僚机启动IFAWL，离站，选择**联合狩猎|僚机**，输入长机的**局域网地址**，连接将启动

在连接之后，两位指挥官将：

1. 确定知晓长机**从空间站带出**的终焉结
2. 口头讨论出一套**完全不同**的终焉结组合**供僚机使用**
3. 讨论过程中，僚机输入**q/w/e**进行选择
4. 最后确保长机僚机指挥官的终焉结**完全不同**

#### 关于联合狩猎 - 战斗

相比于一般战斗，联合狩猎的新知识点如下：

**轮流行动** - 当骰子确定为我方航行日，两位指挥官将各行动一次

**敌方攻击目标** - 敌方目前攻击的我方舰船，其护盾上方将显示\@符号

**一次濒死** - 当其中一位指挥官舰船首次阵亡时，将进入无法行动的濒死状态。任务将继续。此时，另一位指挥官必须在濒死倒计时结束前将其拉起，否则其舰船将在任务中永久损毁。

**濒死倒计时** - 陷入濒死状态的指挥官，护盾上方将显示(+++++)标志。圆括号内的加号代表倒计时，随着任务日数而减小。

**濒死拉起** - 未濒死的指挥官将濒死的舰船拉起时，将把濒死舰船的护盾回补至1。**注意** 回补过程中，濒死舰船为了使护盾达到1所增加的护盾，将要从存活的舰船上扣除。例如，将护盾为-1的濒死舰船拉起，将要使救援者受到2护盾扣减惩罚。

**二次死亡** - 被濒死拉起后的舰船，护盾上方将显示(-)标志。此时，若舰船再次阵亡，将直接进入死亡状态，不能再行动或被拉起。

### 终焉结平衡性调整

#### 关于 “瞳猫”反应堆尾流推进器#27

**缘由**-瞳猫长充能、低护盾的闪避需求对于绝大多数战斗场景来说都是难以满足的，这使得瞳猫的工作场景仅限于少数几个特定的配队。因此，我们希望通过增加功能的方式拓展瞳猫的就业面。

**增强**-瞳猫将为使层数清零的攻击提供150%向下取整的增伤。

#### 关于“风间浦”全平台战术安全港#34

**缘由**-风间浦的保守模式几乎没有使用场景。几乎所有指挥官都会在冷却结束的瞬间开启激进模式开始搏杀，而不是使用保守模式进行回盾。

**增强**-提高保守模式的回盾加强概率。为冷却引入回盾加强。削弱了激进模式。

### 不正常现象
这些现象是**不正常**的。遇到它们时，请联系我们。
- 任何游戏崩溃
- 终焉结在两次战斗之间未能重置自身状态，如冷却冷到下一把开始还没冷却完
- 仓库物品数量异常，出现负数或不能正常增减
- 战斗过程中出现未携带的终焉结的话语
- 舰船名字是空白或无法正确命名舰船
- 自动驾驶出现错误操作
- 终焉结行为与描述不匹配（如大奶油的小奶油偷到导弹之后不能爆炸）
- 联机模式下出现终焉结、护盾导弹行为异常
